1"""Core methylation extraction functions for bam2tensor. 

2 

3This module contains the main function for extracting methylation data from 

4BAM files: extract_methylation_data_from_bam(). This function reads aligned 

5bisulfite-sequencing reads and generates a sparse matrix representation of 

6methylation states at each CpG site. 

7 

8The extraction process: 

9 1. Iterates through each chromosome in the genome embedding 

10 2. For each aligned read that passes quality filters: 

11 a. Identifies which CpG sites fall within the read's alignment 

12 b. Determines the bisulfite strand (forward or reverse) 

13 c. Extracts methylation state at each covered CpG 

14 3. Builds a sparse COO matrix with the results 

15 

16Methylation State Encoding: 

17 - 1: Methylated (cytosine remained as C after bisulfite treatment) 

18 - 0: Unmethylated (cytosine converted to T by bisulfite treatment) 

19 - -1: No data (indel, SNV, or other non-C/T base at CpG position) 

20 

21Supported Aligners: 

22 The module supports BAM files from aligners that provide strand information: 

23 - Biscuit: Uses the YD tag (f=forward, r=reverse) 

24 - gem3/Blueprint: Uses the XB tag (C=forward, G=reverse) 

25 

26Example: 

27 >>> # xdoctest: +SKIP 

28 >>> from bam2tensor.embedding import GenomeMethylationEmbedding 

29 >>> from bam2tensor.functions import extract_methylation_data_from_bam 

30 >>> 

31 >>> # Create genome embedding 

32 >>> embedding = GenomeMethylationEmbedding( 

33 ... genome_name="hg38", 

34 ... expected_chromosomes=["chr1", "chr2"], 

35 ... fasta_source="reference.fa", 

36 ... ) 

37 >>> 

38 >>> # Extract methylation data 

39 >>> matrix = extract_methylation_data_from_bam( 

40 ... input_bam="sample.bam", 

41 ... genome_methylation_embedding=embedding, 

42 ... quality_limit=20, 

43 ... ) 

44 >>> 

45 >>> print(f"Extracted {matrix.shape[0]} reads, {matrix.nnz} data points") 

46"""

63 """Extract read-level CpG methylation data from a BAM file. 

64 

65 Parses a bisulfite-sequencing BAM file and extracts methylation states 

66 at each CpG site covered by aligned reads. Returns a sparse matrix where 

67 each row represents a unique read and each column represents a CpG site. 

68 

69 The function applies several filters to reads: 

70 - Mapping quality must be >= quality_limit (default 20) 

71 - Duplicate reads are excluded 

72 - QC-failed reads are excluded 

73 - Secondary and supplementary alignments are excluded 

74 - Only reads on the bisulfite-converted parent strand are processed 

75 

76 For each qualifying read, the function: 

77 1. Uses binary search to find CpG sites within the read's alignment 

78 2. Checks the bisulfite strand tag (YD or XB) to determine strand 

79 3. Extracts the base at each CpG position from the read sequence 

80 4. Records methylation state: C=methylated(1), T=unmethylated(0), other=-1 

81 

82 Args: 

83 input_bam: Path to the input BAM file. The file must be indexed 

84 (have a corresponding .bam.bai file). 

85 genome_methylation_embedding: A GenomeMethylationEmbedding object 

86 that defines the CpG site positions and provides coordinate 

87 conversion methods. 

88 quality_limit: Minimum mapping quality (MAPQ) threshold for reads. 

89 Reads with MAPQ below this value are skipped. Default is 20, 

90 which excludes reads mapping to multiple locations equally well. 

91 verbose: If True, display a progress bar and print the total read 

92 count. Useful for monitoring progress on large files. 

93 debug: If True, enable extensive validation and debug output. 

94 Prints per-read information and validates that each read is 

95 only processed once. Significantly slower. 

96 

97 Returns: 

98 A scipy.sparse.coo_matrix with shape (n_reads, n_cpg_sites) where: 

99 - n_reads is the number of reads that passed filters and covered 

100 at least one CpG site 

101 - n_cpg_sites is genome_methylation_embedding.total_cpg_sites 

102 - Values are: 1 (methylated), 0 (unmethylated), -1 (no data) 

103 

104 The matrix uses COO format for efficient construction. Convert to 

105 CSR (tocsr()) for row slicing or CSC (tocsc()) for column slicing. 

106 

107 Raises: 

108 FileNotFoundError: If the BAM file index (.bam.bai) is missing. 

109 The BAM file must be indexed with `samtools index`. 

110 

111 Example: 

112 >>> # xdoctest: +SKIP 

113 >>> from bam2tensor.embedding import GenomeMethylationEmbedding 

114 >>> from bam2tensor.functions import extract_methylation_data_from_bam 

115 >>> import scipy.sparse 

116 >>> 

117 >>> # Create embedding (or load from cache) 

118 >>> embedding = GenomeMethylationEmbedding( 

119 ... genome_name="hg38", 

120 ... expected_chromosomes=["chr1", "chr22"], 

121 ... fasta_source="GRCh38.fa", 

122 ... ) 

123 >>> 

124 >>> # Extract methylation data 

125 >>> matrix = extract_methylation_data_from_bam( 

126 ... input_bam="sample.bam", 

127 ... genome_methylation_embedding=embedding, 

128 ... quality_limit=30, # Stricter quality filter 

129 ... verbose=True, 

130 ... ) 

131 >>> 

132 >>> # Analyze results 

133 >>> print(f"Reads with CpG data: {matrix.shape[0]:,}") 

134 >>> print(f"Total CpG sites: {matrix.shape[1]:,}") 

135 >>> print(f"Data points: {matrix.nnz:,}") 

136 >>> 

137 >>> # Save to file 

138 >>> scipy.sparse.save_npz("sample.methylation.npz", matrix) 

139 

140 Note: 

141 The function processes chromosomes in the order they appear in 

142 genome_methylation_embedding.cpg_sites_dict. Reads are numbered 

143 sequentially as they are encountered across all chromosomes. 

144 

145 See Also: 

146 GenomeMethylationEmbedding: For creating the genome embedding. 

147 scipy.sparse.coo_matrix: Documentation on the output format.
